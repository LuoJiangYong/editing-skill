## 一、事件触发：OpenAI 再度大幅降价

2026年6月17日，OpenAI 宣布 GPT-5o-mini 价格下调 40%，speech 系列改按字符计费。这是 OpenAI 在 2026 年的第三次降价。同日，国内多家厂商跟进降价，部分降幅达 30%-50%。

这不是孤立事件。如果把时间轴拉长，自 2024 年以来，AI API 价格呈现持续、加速的下降趋势。

## 二、价格走势：两年降 87% 的速度

### 2.1 数据回顾

从 GPT-4 到 GPT-5o-mini，同等能力级别的 API 价格从 $60/M tokens 降至 $8/M tokens，降幅 87%。这个速度甚至超过了摩尔定律（18 个月性能翻倍）。

推动因素：
- 模型架构优化（MoE、量化、蒸馏）
- 推理硬件进步（专用芯片、更大规模集群）
- 竞争压力（开源模型、新进入者）

### 2.2 降价的三波浪潮

| 阶段 | 时间 | 驱动力 | 降价幅度 |
|------|------|-------|---------|
| 第一波 | 2024 | 规模效应 + 架构优化 | 30-50% |
| 第二波 | 2025 | 开源模型竞争 | 50-70% |
| 第三波 | 2026 | 多厂商价格战 | 30-40% 加速中 |

> ⚠️ 以上数据为示例虚构。实际生产时需从各厂商公开定价页获取真实数据。

## 三、竞争格局：开源模型的"鲶鱼效应"

### 3.1 定价对比

| 模型类型 | 代表 | 价格（$/M tokens） | 相对 GPT-5o |
|---------|------|-------------------|------------|
| 闭源旗舰 | GPT-5o | $15 | 基准 |
| 闭源轻量 | GPT-5o-mini | $8（降价后） | 53% |
| 开源部署 | Llama 4 系列 | $3（推理成本） | 20% |
| 开源免费 | 本地运行 | $0（仅算力） | 0% |

> ⚠️ 以上价格为示例虚构。

### 3.2 开源模型的压力传导机制

开源模型通过三条路径影响闭源定价：

1. **直接替代**：开发者将推理负载从闭源 API 迁移到开源自部署方案
2. **议价锚点**：开源方案的成本成为企业谈判的参考锚点
3. **能力追赶**：开源模型质量逐步逼近闭源，削弱闭源的差异化优势

## 四、影响分析

### 4.1 对开发者的影响

AI 应用开发成本持续下降。一个典型的 AI-native 产品在 2024 年的 API 成本可能占总运营成本的 40%，到 2026 年可能降至 10% 以下。这释放了预算用于产品创新而非基础设施。

### 4.2 对创业公司的影响

价格战是双刃剑：低成本降低了创业门槛，但也意味着"API 套壳"类产品的竞争壁垒进一步降低。

### 4.3 对 AI 厂商的影响

降价→用户增长→更多数据→模型改进→成本再降→再降价。这个飞轮一旦转起来，规模不足的厂商将被挤出。

## 五、前景判断

基于当前趋势的编辑判断（非预测）：

1. **降价不可逆**：开源模型的持续进步意味着闭源模型的定价权将持续被侵蚀。
2. **差异化转向**：单纯卖 API token 的商业模式不可持续。厂商将向更高层（应用、平台、生态）迁移。
3. **关注变量**：如果出现突破性技术（如全新架构），可能暂时打破降价趋势——但长期看开源仍会追上来。
