### 1. OpenAI 调整 GPT-5o 系列 API 定价

OpenAI 宣布即日起调整 GPT-5o 系列 API 定价。GPT-5o-mini 价格下调 40%，GPT-5o-speech 新增按字符计费模式。这一调整被普遍认为是对近期开源模型竞争加剧的回应。

> 📎 来源：[^ref-001] OpenAI 官方博客（6月17日）

### 2. Meta 开源 VLM-3.1 视觉模型更新

Meta 发布 VLM-3.1 系列更新，在文档理解基准上比前代提升 15%。采用 Apache 2.0 许可证，已在 Hugging Face 上架。7B、13B、34B 三个规模同步发布。

> 📎 来源：[^ref-002] Meta AI Blog（6月17日）

### 3. 字节跳动发布"即梦 AI"视频编辑工具

字节跳动正式对外发布 AI 视频编辑工具，支持文本生成视频片段、智能剪辑和风格迁移。目前面向创作者开放公测，暂未公布正式定价。

> 📎 来源：[^ref-003] 字节跳动官方公告（6月17日）

### 4. HuggingFace 平台日活模型下载量破 10 亿

HuggingFace 宣布平台日活模型下载量首次突破 10 亿次，相比半年前增长 3 倍。开源模型的采用速度正在加速。

> 📎 来源：[^ref-004] HuggingFace Blog（6月17日）

### 5. Anthropic 发布 Claude 4 安全评测报告

Anthropic 发布 Claude 4 系列独立安全评测报告，涵盖越狱抵抗、偏见检测、误用风险评估等维度。报告显示在多项安全基准上相比前代有显著提升。

> 📎 来源：[^ref-005] Anthropic Blog（6月17日）

### 6. 国内多家厂商跟进 API 降价

受 OpenAI 降价影响，国内多家大模型厂商宣布 API 价格调整。部分厂商降幅达 30%-50%，API 价格战持续升温。

> 📎 来源：[^ref-006] 综合多家科技媒体报道（6月17日）

### 7. GitHub Copilot 新增多模型支持

GitHub Copilot 宣布支持切换底层模型。开发者可在 Claude 3.5 Sonnet、Gemini 2.0 和 GPT-5o 之间自由选择。该功能即日起逐步推送。

> 📎 来源：[^ref-007] GitHub Blog（6月16日，昨夜发布）
